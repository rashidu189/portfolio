Car Park Schedule Automation is an RPA solution developed to automate the process of retrieving parking schedules from a website using location data provided in an Excel file. This automation eliminates manual searching and data entry, improving efficiency and accuracy.

Demo Video:

Technologies Used:

  • UiPath (RPA Development)
  • Excel Automation
  • Web Scraping / Data Extraction

Process Overview:

  • Read Excel File: The bot reads car park location details from an input Excel file.
  • Search Location: Each location is searched on the target website automatically.
  • Data Extraction: Relevant parking schedules are extracted from the website.
  • Write to Excel: Extracted data is written back into the Excel file.
  • File Renaming: The Excel file is renamed based on the corresponding car park location.

Key Features:

  • Automated Web Interaction: Mimics human actions to navigate and search websites.
  • Accurate Data Extraction: Captures structured schedule data without errors.
  • Batch Processing: Handles multiple car park locations in one run.
  • Dynamic File Handling: Automatically renames output files based on data.
  • Error Handling: Handles missing locations or website errors gracefully.

Benefits:

  • Reduces manual effort and saves time
  • Improves data accuracy
  • Ensures consistency in reporting
  • Scalable for large datasets

This automation demonstrates how RPA can be used to streamline repetitive web-based tasks, making business processes faster, more reliable, and highly efficient.
